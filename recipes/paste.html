Paste

Description

Aici avem niste paste smechere si mi-e o pofta de mor.

Ingredinte:

  • Paste
  • Sos de tomate
  • Apa
  • Sare
  • Ulei

Pasi:

  1. Se pune apa pe foc.
  2. Se adauga o lingura de ulei.
  3. Se adauga sare dupa gust.
  4. Cand incepe apa sa fiarba se adauga pastele.
  5. Dupa 10 minute se scurge apa si se adauga sos de tomat peste paste
